# IEOP API V2

## Setup

```bash
npm install
```

Cria o teu `.env` a partir do exemplo:

```bash
copy .env.example .env
```

## Correr

Dev (com reload):

```bash
npm run dev
```

Produção:

```bash
npm start
```

Healthcheck:

- GET `http://localhost:3000/health`

Swagger:

- `http://localhost:3000/docs`
